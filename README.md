# 📄 AI Resume Analyzer & Job Matcher

This is a complete, production-ready Python web application that analyzes uploaded resumes (PDF/DOCX), generates a professional performance rating, and recommends ideal companies and job profiles using optimized text processing matching.

## 🚀 Features
- **Instant Resume Upload:** Support for `.pdf` and `.docx` file formats.
- **Automated Score Generation:** Analytical feedback and rating scale (0-100).
- **Targeted Corporate Matching:** Real-time job profile and major company alignment suggestions.
- **Optimization Suggestions:** Directly highlights missing focus points to boost resume selection rates.

## 💻 How to Run Locally
1. Install requirements: `pip install -r requirements.txt`
2. Run the application: `streamlit run app.py`
